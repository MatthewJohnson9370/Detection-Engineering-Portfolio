# Detection Engineering Portfolio

Welcome! I'm an InfoSec IAM Analyst with a background in offensive security, building a portfolio focused on behavior-based detection engineering and threat hunting.

This repository documents my journey researching adversary tradecraft, emulating real-world attacks, and developing detections that focus on attacker **TTPs** rather than simple indicators of compromise. Each project includes the attack methodology, collected telemetry, detection logic, validation, tuning considerations, and investigation guidance.

## Methodology

**Research → Build Lab → Execute Attack → Analyze Telemetry → Develop Detection → Validate → Tune → Document**

My goal is to build high-fidelity detections that identify attacker behavior and remain effective regardless of the specific tools used.
